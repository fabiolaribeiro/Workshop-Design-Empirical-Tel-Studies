# Design Empirical TEL Studies in Higher Education
Practical Research Insights from the CORE Project (JTEL)

This repository contains workshop materials and fictional longitudinal TEL examples developed for interactive discussions about:

- longitudinal research design
- intervention studies
- assessment strategies
- multimodal data
- mixed-method analyses
- evidence generation in TEL
